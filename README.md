# Writer's Room (`wroom`)

A fiction studio disguised as a social network. You run a **room** of invented
characters, step fully into one at a time, and live inside that character's
social world — their followers, the people they follow, their timeline, their
voice. **One author, many identities.**

This is unambiguously a tool for **fiction**. Every character is invented and
every post is fictional craft.

## Stack

- **Vite + React + TypeScript** single-page app, mobile-first (iPhone).
- **Capacitor** wraps the web app as a native iOS app.
- **No backend** — state lives in the browser (`localStorage`) with full JSON
  export/import so a room survives reloads and can be moved between devices.

## Run on the web (dev)

```bash
npm install
npm run dev
```

## Run as an iPhone app

```bash
npm install
npm run ios     # builds web assets, syncs Capacitor, opens Xcode
```

Then press ▶ in Xcode targeting a simulator or a connected device. The iOS
native project lives in `ios/` (generated by `npx cap add ios`).

## Project layout

```
src/
  types.ts          domain model (Author, Character, WorldAccount, Follow, Post)
  store/            persistence + React state, seed room
  lib/              avatars, relative time, color helpers
  components/       composer, post card, identity chip, command palette, tab bar…
  screens/          auth, room, character editor/home/profile, post detail, settings
```

## Fiction safeguards

A persistent “Fiction” tag rides on profiles and exports; shareable views carry
a clear *“Fictional content — created in Writer's Room”* watermark. The app never
mimics a real platform's name, logo, or verified-badge iconography.
